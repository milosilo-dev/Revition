# Revition
Install for windows:
. install python from www.python.org
. run in terminal: pip install flask

run on windows:
. python main.py
. goto 127.0.0.1:5000

Install for Ubuntu:
. install python pip from the commands : sudo apt update, sudo apt install python3-pip
. then pip install flask

run on ubuntu:
. python3 main.py
. goto 127.0.0.1:5000
